[Georgiev, D., Fernández-Galiana, Á., Pedersen, S., Papadopoulos, G., Xie, R., Stevens, M., & Barahona, M. (2024). Hyperspectral unmixing for Raman spectroscopy via physics-constrained autoencoders.](https://arxiv.org)
